# WHO Growth Standards Data

`standards.json` is a compact local extract from official WHO Child Growth Standards expanded percentile tables.

Included dashboard curves:

- P3
- P50
- P97

Included indicators:

- Weight-for-age
- Length/height-for-age
- Head circumference-for-age

Included sexes:

- Female
- Male

The dashboard uses this data for visual reference only. BabyLog is not a medical device and does not provide medical advice.

Source pages:

- https://www.who.int/tools/child-growth-standards/standards/weight-for-age
- https://www.who.int/tools/child-growth-standards/standards/length-height-for-age
- https://www.who.int/tools/child-growth-standards/standards/head-circumference-for-age
